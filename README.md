# MarsRover

MarRover is a command line application.

## Prerequisites

You will need `composer` installed to your system.

## Usage

Checkout repository:

```
```

Make sure all dependencies are installed:

```
cd MarsRover
composer install
```

To start the server run:

```
./index.php
```
